# Apex Implement - Phase d'Implémentation

Troisième phase: exécuter le plan tâche par tâche.

## Prérequis

Avoir fait `/apex-analyze` et `/apex-plan`.

## Processus

### 1. Charger le Plan

- Lire `/spec/tasks/[feature].md`
- Vérifier les dépendances

### 2. Exécuter les Tâches

Pour chaque tâche:
1. Lire la description
2. Implémenter le code
3. Vérifier TypeScript
4. Marquer comme complete dans le fichier

### 3. Points de Contrôle

À chaque checkpoint:
- `npm run build`
- `npm run lint`
- Test manuel si applicable

### 4. Finalisation

- Résumer les changements
- Lister les fichiers modifiés
- Suggérer les prochaines étapes
