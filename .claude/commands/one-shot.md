# One-Shot Task

Execute une tâche simple et autonome en une seule passe.

## Quand Utiliser

- Bug fixes simples
- Ajout de composants UI isolés
- Refactoring mineur
- Ajout de types/interfaces
- Mise à jour de textes

## Processus

1. **Comprendre** - Analyser la demande
2. **Identifier** - Trouver les fichiers concernés
3. **Implémenter** - Coder la solution
4. **Vérifier** - `npm run build`
5. **Résumer** - Lister les changements
