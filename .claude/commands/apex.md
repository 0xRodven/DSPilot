# Apex Task - Implémentation Structurée

Pour les tâches de complexité moyenne nécessitant un plan.

## Processus

### 1. Analyse (2 min)

- Lire la demande et les specs
- Identifier les dépendances
- Lister les fichiers à modifier

### 2. Plan (3 min)

- Définir les étapes
- Estimer la complexité
- Identifier les risques

### 3. Implémentation (variable)

- Suivre le plan étape par étape
- Tester après chaque étape

### 4. Vérification (2 min)

- `npm run build`
- Test manuel si possible
