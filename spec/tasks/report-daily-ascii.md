# Daily Report ASCII v1

```text
DAILY REPORT | STATION | YYYY-MM-DD

1. Headline
   - One sentence on the operational state of the day
   - Mention whether there are new qualified alerts

2. KPI strip
   - DWC
   - IADC
   - Risque DNR
   - Drivers actifs

3. Highlights
   - What changed since the previous daily snapshot
   - Which defect type dominates
   - Whether the data freshness is acceptable

4. Priorités du jour
   - Top coaching priority
   - Top operational watchpoint
   - Mapping or data issue if any

5. Drivers en amélioration
   - Top 3 to 5 drivers

6. Drivers à risque
   - Bottom 3 to 5 drivers

7. Alertes qualifiées
   - Severity
   - Summary
   - Link target in DSPilot

8. Traçabilité
   - Generated at
   - Latest data timestamp
   - Confidence label
```
